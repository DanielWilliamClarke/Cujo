# modern-cv-app
A Dockerised CV/Portfolio application built using a MGRN (Mongodb, Golang, React, Node) Technology stack.

The app is arranged into a small microservice architecture.

# Microservices

**Client** - React / Webpack 

**Server** - Golang

**Storage** - MongoDB / Minio
